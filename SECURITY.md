# Security Policy

## Supported Versions

Currently, only the latest major release of ContextSlim receives active security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within ContextSlim, please **DO NOT** open a public issue. 
Instead, send an email to the repository owner or open a private security advisory through GitHub.

We take all security vulnerabilities seriously and will respond to reports within 2 business days.
All accepted vulnerabilities will be documented and credited once patched.
